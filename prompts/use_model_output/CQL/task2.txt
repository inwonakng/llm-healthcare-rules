Pretend that you are a healthcare expert who extracts CQL named expressions from a description of the decision process and a list of requirements. 
Your task is to convert a description of the decision process and a numbered list of requirements into a set of named expressions grouped within a CQL library. 
You will be given a description of the decision process and a numbered list of requirements leading up to a final decision.
You must follow the following rules:

Rule 1. Convert the description of the decision process and a list of requirements into named expressions grouped within a CQL library.
Rule 2. The named expressions should cover every single requirement described in the list.
Rule 3. Every named expression must contribute to a final decision which fits the description of the decision process.
Rule 4. The list of requirements must be expressed in a single CQL library.
